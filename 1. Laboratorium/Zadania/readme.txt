*******
* C++ *
*******
W celu skompilowania kodu programu (w systemie Linux) 
napisanego w języku C++, należy przejść do katalogu 
Cpp i wykonać w terminalu polecenie: 

make

Aby uruchomić skompilowany program, należy wykonać 
polecenie: 

./main

**********
* Python *
**********
Kod napisany w języku Python znajduje się w pliku
MOwNiT – Arytmetyka Komputerowa.ipynb w katalogu Python.
Plik ten jednocześnie zawiera opracowanie wyników obliczeń
wykonanych w języku C++ oraz w języku Python. Wyeksportowany
plik do pliku pdf znajduje się w bieżącym katalogu.

Najlepszym sposobem uruchomienia pliku 
MOwNiT – Arytmetyka Komputerowa.ipynb jest zainstalowanie
Jupyter Notebooka. Aby móc zainstalować Jupyter Notebooka,
konieczna jest uprzednia instalacja języka Python.
Po instalacji, możemy wykonać polecenie:

pip install notebook

Po zainstalowaniu Jupyter Notebooka, uruchamiamy go poleceniem

jupyter-notebook <ścieżka>

gdzie <ścieżka> jest ścieżką do katalogu, w którym ma zostać
uruchomiony notes. Po uruchomieniu, w przeglądarkowym interfejsie
graficznym, możemy wybrać plik notesu, który należy otworzyć.
